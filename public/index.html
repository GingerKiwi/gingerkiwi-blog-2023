Happy Coding (With Cats)!

Welcome to the Ginger Kiwi Blog! This site is developed with 11ty/Eleventy, Nunjucks, markdown, Obsdian, and css with css variables for colours and fonts - and of course lots of "help" from Boo (my grey cat), as well as Maybel and Kitten. Scroll down to see some of my recent and featured articles, or go the the blog page to see all my articles.

Updates for Fall 2023

Nov 13, 2023: Visit https://gingerkiwi-2024-02.fly.dev to see the new combined, fullstack, Ginger Kiwi blog and dev portfolio being built in public on Fly.io using Astro, Docker, and SQLite3!
A static version is also deployed on https://gingerkiwi.xyz.
My ".xyz" domain is a temporary one I have for testing.

It's past time for a redesign of the Ginger Kiwi blog. Having dedicated pages for topics such as French for Devs, design tips for backend devs, DSA, etc, would make the site more user friendly. I also need to set up an RSS, implement some A11y enhancements, fix the styling of article main images, and update the theme to at least a dark mode, and ideally a dark/light toggle. Also, 11ty 2.0 came out months ago, and Astro has been taking off. So I've been on the fence if I'm going to migrate to 11ty 2.0, or try something new with Astro.

This morning (Tuesday, Nov 07), I saw that Lazar Nikolov has a new Astro blog course that's free on Egghead.io for the next two weeks. So here we go! I've started working on a new site that consolodates this blog with my .dev portfolio site.
